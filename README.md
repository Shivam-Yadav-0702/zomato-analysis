# 🍽️ Zomato Indian Restaurants — EDA

## 📌 Objective
Analyze Zomato restaurant data to identify key factors that drive
restaurant success, measured by ratings.

## 📊 Analysis Areas
- Data Overview & Cleaning
- Rating Distribution & Basic Statistics
- City-wise Restaurant Concentration & Ratings
- Most Popular Cuisines & Their Rating Impact
- Price Range vs Rating Analysis
- Impact of Delivery / Table Booking on Ratings
- Top Restaurant Chains in India
- Features Analysis (Wi-Fi, Alcohol, AC, Outdoor)
- Word Cloud from Restaurant Highlights

## 🛠️ Tools & Libraries
| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas & NumPy | Data manipulation |
| Matplotlib & Seaborn | Static visualizations |
| Plotly | Interactive charts |
| WordCloud | Text visualization |

## 💡 Key Insights
- Rating distribution is left-skewed — most restaurants score between 3.5–4.5
- Online delivery & table booking positively impact ratings
- Budget restaurants dominate in count but Luxury ones score higher

## 📁 Dataset
Indian Restaurants dataset from Zomato

## 🚀 How to Run
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Open `Zomato_Indian_Restaurants_EDA.ipynb` in Google Notebook
